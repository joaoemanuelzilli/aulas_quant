# Aula 1 — Introdução ao Python para Finanças

## Objetivos
- Revisar sintaxe básica do Python com foco em finanças
- Operar arrays com NumPy de forma vetorizada
- Calcular e visualizar séries de preços
- Implementar uma simulação de Monte Carlo simples

## Arquivo principal
`aula_01.py` — execute com:
```bash
python aula_01.py
```

## Conceitos cobertos
| Conceito | Descrição |
|----------|-----------|
| `np.random.normal` | Geração de retornos aleatórios com distribuição normal |
| `np.cumprod` | Reconstrução de preços a partir de retornos diários |
| Monte Carlo | Simulação de múltiplos caminhos de preço |
