# Aula 2 — Análise de Dados Financeiros com Pandas

## Objetivos
- Baixar dados históricos com `yfinance`
- Calcular retornos simples e logarítmicos
- Analisar estatísticas descritivas de ativos reais
- Visualizar preços normalizados, retorno acumulado e matriz de correlação

## Arquivo principal
`aula_02.py` — execute com:
```bash
python aula_02.py
```

## Conceitos cobertos
| Conceito | Descrição |
|----------|-----------|
| `yfinance.download` | Download de dados históricos de preços |
| `pct_change()` | Cálculo de retornos simples diários |
| `np.log(P/P_lag)` | Retornos logarítmicos |
| `corr()` | Matriz de correlação entre ativos |
