# Aula 3 — Estatísticas para Finanças

## Objetivos
- Comparar a distribuição de retornos reais com a normal teórica
- Medir assimetria e curtose
- Testar normalidade com Jarque-Bera
- Estimar o modelo CAPM via regressão linear (β do ativo)

## Arquivo principal
`aula_03.py` — execute com:
```bash
python aula_03.py
```

## Conceitos cobertos
| Conceito | Descrição |
|----------|-----------|
| Skewness / Kurtosis | Momentos de 3ª e 4ª ordem da distribuição |
| Teste Jarque-Bera | Teste de normalidade baseado nos momentos |
| Q-Q Plot | Comparação visual com a distribuição normal |
| Regressão CAPM | `retorno_ativo = α + β × retorno_mercado + ε` |
