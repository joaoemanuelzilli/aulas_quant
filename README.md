# Aulas Quant — Núcleo Quant da Liga de Mercado Financeiro

Repositório oficial das aulas do **Núcleo Quant** da Liga de Mercado Financeiro.

---

## 📚 Ementa

| # | Aula | Tópicos |
|---|------|---------|
| 1 | [Introdução ao Python para Finanças](aulas/aula_01_introducao_python/) | Sintaxe básica, NumPy, Matplotlib, séries de preços |
| 2 | [Análise de Dados Financeiros](aulas/aula_02_analise_dados/) | Pandas, yfinance, retornos, estatísticas descritivas |
| 3 | [Estatísticas para Finanças](aulas/aula_03_estatisticas/) | Distribuições, testes de hipótese, correlação, regressão |
| 4 | [Otimização de Portfólio](aulas/aula_04_otimizacao_portfolio/) | Fronteira eficiente de Markowitz, Índice de Sharpe, cvxpy |
| 5 | [Gestão de Risco](aulas/aula_05_gestao_risco/) | VaR histórico e paramétrico, CVaR, backtesting |

---

## ⚙️ Requisitos

Python 3.9+ e os pacotes listados em `requirements.txt`.

```bash
pip install -r requirements.txt
```

---

## 🚀 Como usar

Cada pasta de aula contém:
- Um script `.py` com o código comentado da aula
- Um `README.md` com objetivos e conceitos cobertos

Clone o repositório e execute os scripts com Python:

```bash
git clone https://github.com/joaoemanuelzilli/aulas_quant.git
cd aulas_quant
pip install -r requirements.txt
python aulas/aula_01_introducao_python/aula_01.py
```
